How It Works

Discover the AI-powered process behind CleanTech's waste classification system.

1
Upload Image

Users upload a photo of the waste item through our intuitive web interface.

2
Preprocessing

Image is resized to 224x224 pixels and preprocessed using MobileNetV2 techniques.

3
AI Prediction

Pre-trained CNN analyzes image features using transfer learning for accurate results.

4
Result Display

Predicted class and confidence level are shown instantly with the uploaded image.

Powered by Transfer Learning

Our model leverages pre-trained convolutional neural networks (CNNs) trained on vast datasets, enabling high accuracy with minimal computational resources. This approach classifies waste into Biodegradable, Recyclable, or Trash categories efficiently.
